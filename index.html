About Me

I've been a professional musician until recently, working as a cellist in several symphony orchestras in three countries. This has left me with a very potent set of skills including setting goals, organization, problem solving, self-awareness, and self-direction.

I've decided it's time to change to a new path in life, and to that end I've been learning how to be a web developer. My experience from my career as a musician continues to help me in learning and developing those skills, and I'm eager to continue expanding my knowledge and to gain experience in applying those skills to create the best possible applications for your clients.

Personally, I've been a PC hardware and gaming enthusiast from my earliest days, and I still keep up with developments in the hardware and general PC space. I also enjoy reading, baking bread, collecting vinyl records, Japanese (the language), and even dabble in creative writing.

Featherlite Reddit

A lightweight application built in HTML/CSS/JS that reads the Reddit JSON API and renders subreddit data and comments. I am currently working on creating a new version using React scaffolded with Vite. This is a link to the live site. You can search Reddit, save up to 10 subreddits, it remembers light/dark mode selection. Here is a link to the GitHub repository. Please note: I bear no responsibility for the content on Reddit.

Appointment Planner Project

A simple contact and appointment planner built in React using starter code from Codecademy. My own twist is adding a custom state hook that saves contacts and appointments in local storage and fetches them on page load. Here is a link to the live site. Here is a link to the GitHub repository.

Colour Swatch

A simple site that will display up to five colour bars based on user input. This is a link to the live site. Can accept hexadecimal, RGB, and RGBA values with basic regex checks in place. Text colour changes dynamically based on how light or dark the background colours are; I intentionally avoided using the colour libraries available on NPM as an exercise in basic maths and problem solving, but I make no claim that my solutions are any good. 😏 Here is a link to the GitHub repository.

Dice Roller

Something I put together for games night. This is a link to the live site. Features persistent custom dice rolls that will automatically populate on loading the page. Future long term functionality plans include a backend for saving character sheets and the ability to pick different game systems like d20. Here is a link to the GitHub repository.
