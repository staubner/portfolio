Colour Swatch

A simple site that will display up to five colour bars based on user input. This is a link to the live site. Can accept hexadecimal, RGB, and RGBA values with basic regex checks in place. Text colour changes dynamically based on how light or dark the background colours are; I intentionally avoided using the colour libraries available on NPM as an exercise in basic maths and problem solving, but I make no claim that my solutions are any good. 😏 Here is a link to the GitHub repository.
